# Portfolio Optimization - Time Series Forecasting

Time series forecasting for portfolio management optimization at GMF Investments.

## Project Structure
- `data/processed/` - cleaned and processed datasets
- `notebooks/` - Jupyter notebooks for EDA and modeling
- `src/` - source code modules
- `tests/` - unit tests
- `scripts/` - standalone scripts
